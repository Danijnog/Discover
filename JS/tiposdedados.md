# Tipos de dados
    Conforme o ECMAScript standar temos 9 tipos de dados:

    * Data types
        - Primitive / Primitive value
        - Structural
        - Structural Primitive

## Primitivos
    - String
    - Number
    - Boolean
    - undefined
    - Symbol
    - BigInt
    - Null

## Estruturais
    * Object
        - Array
        - Map
        - Set
        - Date
        - ...
    * Function

## Primitivo estrutural / Structural Root Primitive
    * null
